该代码包含9种第四层L4攻击模式：
1、ATK_VEC_UDP: UDP Plain Flood，发送大量 UDP 数据包，耗尽目标服务器的带宽。
2、ATK_VEC_UDP_PPS: UDP PPS Flood，发送大量小尺寸 UDP 数据包，以高 PPS (每秒数据包数) 攻击目标服务器，消耗其处理能力。
3、ATK_VEC_SYN: TCP SYN Flood，发送大量 TCP SYN 数据包，使目标服务器的 TCP 连接队列溢出，无法处理正常请求。
4、ATK_VEC_STOMP: TCP STOMP Flood，干扰目标服务器的 TCP 连接状态，使其无法正常处理请求。
5、ATK_VEC_ACK: TCP ACK Flood，发送大量 TCP ACK 数据包，消耗目标服务器的资源。
6、ATK_VEC_TCP: TCP Flood，发送大量 TCP 数据包，耗尽目标服务器的带宽和连接资源。
7、ATK_VEC_OVH: UDP OVH Hex Flood，针对 OVH 防御机制的 UDP Flood 攻击，发送特定格式的 UDP 数据包。
8、ATK_VEC_TCPBYPASS: TCP Bypass Flood，尝试绕过一些网络设备的防御机制，直接建立大量 TCP 连接到目标服务器。
9、ATK_VEC_TCPSOCKET: TCP Socket Flood，快速创建大量 TCP Socket 连接到目标服务器，但不发送任何数据，消耗其连接资源。
