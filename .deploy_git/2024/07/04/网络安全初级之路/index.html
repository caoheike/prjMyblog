网络安全初级之路

一、CVE、POC、EXP、REC的 概念

   1、【CVE】: 公开漏洞标识，个人理解为公开漏洞之后的一个编码，根据这个编码可以找到对应漏洞描述信息。

   2、【POC】：理解为用一些数据包完成 漏洞的复现，  EXP一般需要基于poc 完成。

   3、【EXP】：漏洞利用脚本，为某些洞开发的一些脚本。

   4、【RCE】 远程执行的漏洞类型，同类还有 **SQL 注入** 、**跨站请求伪造**、中间人伪造等
